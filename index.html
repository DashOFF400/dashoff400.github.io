Cahier des Charges – Projet de Recensement des Décès

1. Présentation

  • Client : Organismes publics, mairies, morgues et hôpitaux
  • Contexte : Manque de centralisation et de fiabilité des données concernant les personnes décédées.
  • Objectif : Créer une solution numérique pour améliorer le recensement des décès et rendre les données plus accessibles et exploitables.

2. Cibles

  • Utilisateurs visés : Fonctionnaires publics, médecins légistes, agents d’état civil.

3. Fonctionnalités

  • Fonction principale : Ajout d’informations dans une base de données (IFU, nom, prénom, statut, description du décès).
  • Fonction secondaire : Accès public en lecture ; seules les personnes autorisées peuvent modifier ou ajouter des données.
